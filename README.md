# Our Distance - Frontend

Interfaz web para la aplicación de pareja **Our Distance**, construida con Angular.

## Stack

- **Angular** - Framework frontend
- **TypeScript** - Lenguaje principal
- **CSS** - Estilos personalizados

## Requisitos

- Node.js 18+
- Angular CLI

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Mayel-dev/our-distance-app.git
cd our-distance-app

# Instalar dependencias
npm install
```

## Correr el proyecto

```bash
ng serve
```

Visita `http://localhost:4200`

> El backend debe estar corriendo en `http://localhost:3000`

## Páginas

- **Welcome** - Página de bienvenida
- **Register** - Registro de usuario
- **Login** - Inicio de sesión
- **Home** - Dashboard principal
- **Goals** - Gestión de metas (privadas, compartidas, de pareja)
- **Pairing** - Conectar y desconectar pareja
- **Profile** - Ver y editar perfil
